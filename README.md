# HandBoard

An accessibility-focused Android keyboard designed for one-handed use.

## Features (Planned)
- One-handed keyboard layouts (grid, arc, thumb-optimized)
- Left-hand and right-hand modes
- Word prediction (AOSP dictionaries)
- Emoji and symbol support
- Multi-language support
- Fully customizable size, position, and layout

## Build
Push to `main` → GitHub Actions builds the APK automatically.
